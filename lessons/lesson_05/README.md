# Sample Pipeline

## Installations and Set Up

This pipeline requires the tool seqkit!
If you do not have this tool installed, run the install seqkit script from the Lesson 05 directory:

```bash
./scripts/install_seqkit.sh [download location]
```
The tool should now be located in the directory you specified. You can check by running:
```bash
cd [download location]
ls -ahl
```

seqkit should now be there! This directory MUST be in your $PATH to be able to run it from anywhere on your machine!
